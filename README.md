# Image Encryption and Decryption (Python)

This project encrypts and decrypts images using Python.  
It converts an image into an unreadable format and restores it using a secret key.

---

## Features
- Encrypt image
- Decrypt image
- Simple and easy to use

---

## Technologies Used
- Python 3
- Pillow (PIL)

---

## How to Run

Step 1: Install required library
pip install pillow


Step 2: Run the program
python image-encrypter.py
